基于ElementUI的标签页缓存

VUE-ELEMENT-ADMIN

2.15.X，开启Tags的标签页会话级缓存

1. 组件准备

确保项目内存在：

vue-admin-template\src\layout\components\TagsView 文件夹

vue-admin-template\src\store\modules\tagsView.js 文件

如果不存在需要去原项目拷贝

2. 修改AppMain声明缓存页

vue-admin-template\src\layout\components\AppMain.vue

keep-alive新增include属性声明缓存页面

3. 修改index.js载入标签视图组件

vue-admin-template\src\layout\components\index.js

4. 修改index.vue全局视图加入标签组件

vue-admin-template\src\layout\index.vue

5. VUEX引缓存工具

vue-admin-template\src\store\getters.js

vue-admin-template\src\store\index.js

vue-admin-template\src\store\modules\settings.js

6. 动态路由兼容与空路由组判断

vue-admin-template\src\layout\components\TagsView\index.vue

7. 标签页缓存未生效，每次切换都重新加载本页解决

单页中的 vue 声明必须携带 name

且name必须与path拼写一致，但name必须大写，path必须小写

router中的路由声明必须声明keepAlive为true

  • 示例：

router/index.js

对应view

8. KeepAlive声明周期钩子函数

activated：页面第一次进入的时候，钩子触发的顺序是created->mounted->activated
deactivated ：页面退出的时候会触发deactivated，当再次前进或者后退的时候只触发activated
