问题

  • Q: 简称匹配

用户书写习惯的简写方式会忽略区域行政后缀甚至于省市前缀

EXAMPLE：

北京市北京市辖区朝阳区 → 北京朝阳

上海市上海市辖区浦东新区张江镇 → 浦东张江

河南省郑州市中牟县 → 河南郑州中牟县

  • Q:伪补全？

地址逆序匹配 从市区匹配第一个符合规则的即断言其属级赋正确

终止条件为：第一个符合规则的，再优化一点的做法应该是所有符合规则中全称匹配相似度最接近的
