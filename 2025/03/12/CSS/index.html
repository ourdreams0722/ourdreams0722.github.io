概念与API

度量单位

px：表示像素,相对于屏幕分辨率

in：英寸,物理度量文件,在CSS领域被直接映射成像素（1in == 2.54cm == 96px）

cm：厘米,物理度量单位,映射成像素（1cm == 37.8px）

mm：毫米,小数量级的物理度量单位（1mm == 0.1cm == 3.78px ）

pt：点,物理长度单位（1pt == 1/72in == 96/72px）

pc：派卡,物理长度单位（1pc == 12pt）

em：相对单位,相对于当前对象内文本的字体尺寸,字体尺寸未被人设置则相对于浏览器 的默认字体尺寸

rem：和 em一样，不同的是 rem 总是相对于根元素

ex：相对长度单位,此高度通常为字体尺寸的一半,字体尺寸未被人设置则相对于浏览器 的默认字体尺寸当改变 font-family 时 em 不会改变，而 ex 单位会变化

ch：ch的内涵和ex相对于x的高度相似，只是 ch 是基于字符 0 的宽度的而不是基于字符 x 高度的当 font-family 改变的时候 ch 也会随着改变

vw： 可视宽度 单位（视窗宽度），1vw 等于可视区宽度的 百分之一,与百分比类似但它总相对于根元素

vh：可视高度 单位（视窗宽度）,同vw

vmin：vmin的值是当前vw和vh中 较小 的值

vmax：vmax的值是vw和vh中 较大 的那个值

%：百分比,基于父元素

fr：gird布局中利用的一个长度单位，例：声明三行的gird，第一行30px，第二行30px，第三行60px grid-template-rows: 30px 1fr 2fr
